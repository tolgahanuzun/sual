# Sual 

## Requirements:
  - docker
  - docker compose

## Run with:
    $ docker-compose up

* Destroy

      $ ./dev.sh remove

* Container Up

      $ ./dev.sh start

* Bash join

      $ ./dev.sh bash
    
## Reminding:
    $ docker ps -a
    $ docker start id
    $ docker exec -it sual_web_1 bash
